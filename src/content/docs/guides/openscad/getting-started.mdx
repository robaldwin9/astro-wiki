## Install

Popular install methods are listed below. Some AppImage, and nightly build options exist on the [openscad page](https://openscad.org/downloads.html).

<Tabs>
  <TabItem label="Windows">
    ```sh
    winget install --id=OpenSCAD.OpenSCAD -e
    ```
  </TabItem>
  <TabItem label="macOS">
    ```sh
    brew install openscad@snapshot
    ```
  </TabItem>
  <TabItem label="Debian/Ubuntu">
    ```sh
    sudo apt-get install openscad
    ```
  </TabItem>
  <TabItem label="Fedora">
    ```sh
    sudo dnf install openscad
    ```
  </TabItem>
  <TabItem label="Flatpak">
    ```sh
    flatpak remote-add --user --if-not-exists flathub-beta \
      https://flathub.org/beta-repo/flathub-beta.flatpakrepo
    flatpak install --user flathub-beta org.openscad.OpenSCAD
    ```
  </TabItem>
</Tabs>
