## Aside

Asides highlight secondary information alongside the main content.

**Import:**

```mdx
import { Aside } from '@astrojs/starlight/components';
```

<Card title="Default">

**Syntax:**

```mdx
<Aside>Some content in an aside.</Aside>
```

**Result:**

<Aside>Some content in an aside.</Aside>

</Card>

<Card title="Caution">

**Syntax:**

```mdx
<Aside type="caution">Some cautionary content.</Aside>
```

**Result:**

<Aside type="caution">Some cautionary content.</Aside>

</Card>

<Card title="Tip">

**Syntax:**

````mdx
<Aside type="tip">

Other content is also supported in asides.

```js
// A code snippet, for example.
```

</Aside>
````

**Result:**

<Aside type="tip">

Other content is also supported in asides.

```js
// A code snippet, for example.
```

</Aside>

</Card>

<Card title="Danger">

**Syntax:**

```mdx
<Aside type="danger">Do not give your password to anyone.</Aside>
```

**Result:**

<Aside type="danger">Do not give your password to anyone.</Aside>

</Card>

## Tabs

Groups related content into switchable tabs.

**Import:**

```mdx
import { Tabs, TabItem } from '@astrojs/starlight/components';
```

<Card title="Basic tabs">

**Syntax:**

```mdx
<Tabs>
  <TabItem label="Fedora">sudo dnf install package</TabItem>
  <TabItem label="Arch">sudo pacman -S package</TabItem>
</Tabs>
```

**Result:**

<Tabs>
  <TabItem label="Fedora">sudo dnf install package</TabItem>
  <TabItem label="Arch">sudo pacman -S package</TabItem>
</Tabs>

</Card>

<Card title="Tabs with icons">

**Syntax:**

```mdx
<Tabs>
  <TabItem label="Stars" icon="star">
    Sirius, Vega, Betelgeuse
  </TabItem>
  <TabItem label="Moons" icon="moon">
    Io, Europa, Ganymede
  </TabItem>
</Tabs>
```

**Result:**

<Tabs>
  <TabItem label="Stars" icon="star">
    Sirius, Vega, Betelgeuse
  </TabItem>
  <TabItem label="Moons" icon="moon">
    Io, Europa, Ganymede
  </TabItem>
</Tabs>

</Card>
