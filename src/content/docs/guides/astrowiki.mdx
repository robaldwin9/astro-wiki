## Aside

**Syntax:**

```markdown
<Aside>This is an aside.</Aside>

<Aside type="tip">This is a tip.</Aside>

<Aside type="caution">This is a caution.</Aside>

<Aside type="danger">This is a danger warning.</Aside>
```

**Result:**

<Aside>This is an aside.</Aside>

<Aside type="tip">This is a tip.</Aside>

<Aside type="caution">This is a caution.</Aside>

<Aside type="danger">This is a danger warning.</Aside>

## Card & CardGrid

**Syntax:**

```markdown
<Card title="Single Card">
  Content goes here.
</Card>

<CardGrid>
  <Card title="Card 1">First card</Card>
  <Card title="Card 2">Second card</Card>
</CardGrid>
```

**Result:**

<Card title="Single Card">
  Content goes here.
</Card>

<CardGrid>
  <Card title="Card 1">First card</Card>
  <Card title="Card 2">Second card</Card>
</CardGrid>
